# Children-Park

A vibrant and engaging web application designed for a Children's Park. This project serves as a digital gateway for visitors, offering information about park activities, photo galleries, and other essential details.

## Features

- **Interactive Gallery**: Showcase of park highlights and events.
- **Activity Information**: Details on available games, rides, and schedules.
- **Visitor Guidance**: Location, timings, and entry fee information.
- **Responsive Design**: Optimized for viewing on mobile and desktop devices.

## Tech Stack

- **Frontend**: HTML5, SCSS/CSS3, JavaScript
- **Backend**: Python (Scripting/Utilities)

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Rahulbashyal/Children-Park.git
    ```
2.  Open `index.html` (or `index.php`) in your browser to view the site.
